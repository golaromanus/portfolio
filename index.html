How the AI Insight Engine Works

Source DataStructured records & live feeds
→
Rules EngineDeterministic logic decides the outcome
→
LLM LayerOpenAI + Claude behind a provider abstraction
→
ValidationOutput grounded & checked against source — anti-hallucination
→
Insight OutConfidence-tiered, explainable result

The LLM never invents facts — it explains what the rules engine already decided, and every explanation is validated against the source data before it reaches the user.
